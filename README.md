# Django Employee Management System
## Project Description
A simple Employee Management System built using Django.

## Features
- Add Employee
- View Employee
- Upload Employee Image
- Django Admin Panel
- SQLite Database
## Technologies Used
- Python
- Django
- SQLite
- HTML
- CSS
## Installation
Clone the repository
git clone https://github.com/Vamshi-tech1/django-employee-management.git
Install dependencies
pip install -r requirements.txt
Run migrations
python manage.py migrate
Start the server
python manage.py runserver
Open:
http://127.0.0.1:8000/
